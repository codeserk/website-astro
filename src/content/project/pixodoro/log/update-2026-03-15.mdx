The very first version! I've been wanting to build this for a while now, and I finally got around to it.

The core is there: a basic Pomodoro timer with work, break and long-break cycles. Nothing fancy, but it works. I also added session history tracking so you can see how productive you've been (or _haven't_ been).

The fun part was building the pixel art UI. I went for an iPod-style layout with dithered backgrounds and a retro palette. It already looks like a little Game Boy app, which is exactly what I was going for.
